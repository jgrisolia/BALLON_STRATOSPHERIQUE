Using AKERU Early stage Readme.txt

1) download and install Arduino ui or equivalent here :

http://arduino.cc/en/Main/Software

2) donwload and install FTDI drivers here :

http://www.ftdichip.com/Drivers/VCP.htm

3) download and install Akeru library (ie unzip Akeru.zip into the libraries directory)
You'll find an example into the zip file

http://snoot.it/akerulib

4) connect Akeru to your computer and wait for installation

5) using Arduino ui, select the USB port on which Akeru is connected and board type "Arduino Uno"

6) include the Akeru library into the program with instruction : #include <Akeru.h>
